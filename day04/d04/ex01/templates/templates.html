Django template engine

  • Blocks
    1. Make block in base template like this: {% content block %}
    2. Put base content on it.
    3. Close block like this: {% endblock content %}
    4. Extends base template like this: {% extends 'base.html' %}
  • for ... in loop
    1. for loop like this:
    2. <ul>
    3. {% for component on list %}
    4. <li>{{ component }}<li>
    5. {% endfor %}
    6. </ul>
  • Variables through view.py
    1. Make class in models.py
    2. In this class, make class variables
    3. Connect this models to views.py
    4. Can use this variables in template like this : {{ Class.member }}
